クラスとID

クラスとID

前項で属性と値の指定の仕方を解説したが, クラスとIDは例外で, クラスの場合は要素名.クラス名, IDの場合は要素名#ID名の書式で記述できる.
なお, 要素名がdivの場合は要素名も省略可能である.

Pug

#main
  h1.title シシトウ
  p.msg 苦い

これをHTMLに変換すると以下のようになる.

HTML

<div id="main">
  <h1 class="title">シシトウ</h1>
  <p class="msg">苦い</p>
</div>
